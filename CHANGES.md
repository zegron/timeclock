# Changelog

All notable changes to this project will be documented in this file.

## [v1.1.0] - 2025-10-01
### Added
- "View Hours" feature: users can now see **daily**, **weekly**, and **all-time** summaries directly within the app interface.

### Removed
- Removed the **Undo** feature to simplify the workflow and improve reliability.

## [v1.0.0] - 2025-09-24
### Initial Release
- Basic time clock with **Punch In / Out** functionality.
- Automatic CSV logging of time records.
- Simple Tkinter-based user interface.
